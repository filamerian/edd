Linear Regression Analysis

Predicting Relationships Between Variables

Introduction

Linear regression is a statistical method used to predict the value of one variable based on the value of another variable.

It is commonly used in research to model relationships and make predictions.


Learning Objectives

  • Define linear regression
  • Explain dependent and independent variables
  • Compute a regression equation
  • Interpret regression results

Historical Background

Linear regression was developed from the work of Sir Francis Galton and Karl Pearson in the late 19th century.

It was originally used to study biological relationships, such as height inheritance, and later expanded into economics, education, and social sciences.


Definition

Linear regression is a statistical technique that models the relationship between a dependent variable (Y) and an independent variable (X) using a straight line.


Regression Equation

Y = a + bX
  • Y = predicted value
  • a = intercept
  • b = slope
  • X = independent variable

Assumptions

  • Linear relationship between X and Y
  • Homoscedasticity (constant variance)
  • Independence of observations
  • No extreme outliers

Worked Example

Research Scenario

A teacher wants to predict student exam scores based on study hours.

Given Data Summary
  • n = 5 students
  • ΣX = 25
  • ΣY = 350
  • ΣXY = 1850
  • ΣX2 = 135

Solution

Step 1 - Formula for Slope (b)
b = [ nΣXY - (ΣX)(ΣY) ] / [ nΣX2 - (ΣX)2 ]
Step 2 - Substitute Values

b = [ (5 × 1850) - (25 × 350) ] / [ (5 × 135) - (25)2 ]

Step 3 - Compute Numerator

= (9250 - 8750)

Numerator = 500
Step 4 - Compute Denominator

= (675 - 625)

Denominator = 50
Step 5 - Slope (b)
b = 500 / 50 = 10
Step 6 - Intercept (a)
a = (SY / n) - b(SX / n)

a = (350 / 5) - 10(25 / 5)

a = 70 - 10(5)

a = 70 - 50 = 20

a = 20
Step 7 - Regression Equation
Y = 20 + 10X
Step 8 - Interpretation

The slope indicates that for every additional hour of study, the exam score increases by 10 points.

The intercept (20) represents the predicted score when study hours are zero.


Prediction Example

If a student studies for X = 6 hours:

Y = 20 + 10(6) = 80

The predicted exam score is 80.
