Z-Test for Proportions

Comparing Sample Proportion with Population Proportion

Introduction

In research, we often analyze categorical data such as pass/fail, yes/no, or success/failure outcomes. When dealing with proportions, we use the Z-test for proportions to determine whether a sample proportion is significantly different from a known population proportion.

This test is commonly used in education, business, and social science research when outcomes are expressed in percentages or proportions.


Learning Objectives

  • Define the Z-test for proportions.
  • Identify when to use the test.
  • Explain the assumptions of the test.
  • Compute and interpret Z-test results for proportions.

Historical Background

The Z-test for proportions is based on the development of the normal approximation to the binomial distribution. Early statisticians discovered that when sample sizes are large enough, binomial distributions can be approximated using the normal curve.

This approximation allowed researchers to analyze proportions using Z-scores, making hypothesis testing for categorical data more practical and widely applicable.


Definition

The Z-test for proportions is a statistical test used to determine whether a sample proportion (p^) is significantly different from a population proportion (p).


When to Use the Z-Test for Proportions

  • The data is categorical (yes/no, pass/fail, etc.).
  • The sample size is large enough (np ≥ 5 and n(1-p) ≥ 5).
  • The population proportion is known or assumed.
  • Observations are independent.

Z-Test Formula for Proportions

Z = (p^ - p) / sqrt( p(1 - p) / n )

Where:

  • p^ = sample proportion
  • p = population proportion
  • n = sample size

Assumptions

  • The sample is randomly selected.
  • Observations are independent.
  • Sample size is large enough for normal approximation.
  • Data is categorical (binary outcomes).

Worked Example

Research Scenario

A researcher wants to determine whether the pass rate of students in a statistics exam is different from the known university pass rate.

The university claims that 70% of students pass the exam. The researcher collects a sample of students to verify this claim.

Given Information
  • Population Proportion (p) = 0.70
  • Sample Proportion (p^) = 0.78
  • Sample Size (n) = 50

Solution

Step 1 - State the Hypotheses

We begin by stating the null and alternative hypotheses.

  • Null Hypothesis (H0): There is no significant difference between the sample proportion and the population proportion.
  • Alternative Hypothesis (H1): There is a significant difference between the sample proportion and the population proportion.
H0: p^ = p
H1: p^ != p
Step 2 - Identify the Formula

The Z-test formula for proportions is:

Z = (p^ - p) / sqrt( p(1 - p) / n )
Step 3 - Substitute the Given Values

Given:

  • p^ = 0.78
  • p = 0.70
  • n = 50

Substitute into the formula:

Z = (0.78 - 0.70) / sqrt( (0.70)(1 - 0.70) / 50 )

Step 4 - Compute the Values

Compute numerator:

0.78 - 0.70 = 0.08

Compute denominator:

(1 - 0.70) = 0.30
(0.70)(0.30) = 0.21
0.21 / 50 = 0.0042
sqrt(0.0042) = 0.0648 (approx)

Now compute Z:

Z = 0.08 / 0.0648

Z = 1.23 (approx)

Step 5 - Decision Rule

At a = 0.05 (two-tailed test), the critical value is:

Z critical = ±1.96

Decision rule:

  • If |Z computed| > 1.96, reject H0
  • If |Z computed| = 1.96, fail to reject H0
Step 6 - Decision
Z = 1.23

Since 1.23 is less than 1.96, we fail to reject the null hypothesis.

Step 7 - Interpretation

There is no sufficient evidence to conclude that the student pass rate is significantly different from the university's claimed pass rate of 70%.

This suggests that the observed sample proportion is statistically consistent with the population proportion.


Applications

  • Exam pass rate evaluation
  • Survey response analysis (yes/no outcomes)
  • Quality control defect rates
  • Voting and polling studies

Summary

The Z-test for proportions is used to compare a sample proportion with a known population proportion. In this example, the computed Z value (1.23) was less than the critical value (±1.96), leading to a failure to reject the null hypothesis.
